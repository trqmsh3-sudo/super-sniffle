## 🔄 זרימת עבודה מלאה

### **תרחיש: משתמש מחפש מוצר "iPhone 15"**

#### **שלב 1: חיפוש בדאטאבייס** 🔍

```
משתמש → Frontend (/search)
   ↓
POST /api/search?q=iPhone 15
   ↓
Backend בודק בדאטאבייס:
   SELECT * FROM products WHERE name ILIKE '%iPhone 15%'
   ↓
תוצאה: נמצא / לא נמצא
```

**קוד:** `backend/server-unified.js` - `GET /api/search`

---

#### **שלב 2: בניית דוסייה חדשה** 🏗️

אם המוצר לא נמצא (או שיש דוסייה לא מלאה):

```
Frontend → POST /api/products/build
{
  "productName": "iPhone 15",
  "category": "smartphones"
}
   ↓
Backend: SimpleDossierBuilder.buildDossier()
   ↓
┌─────────────────────────────────────────────┐
│ שלב 2.1: AI Analysis (Claude Web Search)   │
└─────────────────────────────────────────────┘
   ↓
Claude עם Web Search Tool:
   • מחפש ב-Reddit (site:reddit.com iPhone 15 review)
   • מחפש ב-YouTube reviews
   • מחפש ב-Amazon reviews
   • מחפש בבלוגים טכנולוגיים
   ↓
Claude מחזיר JSON:
{
  "summary": "...",
  "pros": [...],
  "cons": [...],
  "common_issues": [...],
  "best_for": [...],
  "not_for": [...],
  "overall_sentiment": "positive",
  "value_rating": "good",
  "confidence": 85
}
   ↓
┌─────────────────────────────────────────────┐
│ שלב 2.2: חישוב ציונים                      │
└─────────────────────────────────────────────┘
   ↓
calculateScores(aiData):
   • overall_score (0-100)
   • quality_score (0-100)
   • value_score (0-100)
   • reliability_score (0-100)
   ↓
┌─────────────────────────────────────────────┐
│ שלב 2.3: שמירה בדאטאבייס                   │
└─────────────────────────────────────────────┘
   ↓

INSERT INTO dossiers (product_id, scores, analysis, ...)
   ↓
החזרת productId למשתמש
```

**קוד:**
- `backend/services/simpleAI.js` - `buildDossier()` - Claude Web Search
- `backend/services/simpleDossierBuilder.js` - `buildDossier()` - Orchestration
- `backend/server-unified.js` - `POST /api/products/build` - API Endpoint

#### **שלב 3: הצגת התוצאה** 📊

```
Frontend → GET /api/products/:id

החזרת JSON מלא:
{
  "product": { id, name, category },

   • Data Sources
```

**קוד:**
- `backend/services/simpleDossierBuilder.js` - `getDossier()`

## 🤖 מערכת ה-AI

### **Smart AI Routing** 🧠

המערכת משתמשת ב-**2 מודלי AI** בצורה חכמה:

#### **1. Google Gemini 1.5 Flash** 🟢
- **שימוש:** זיהוי מוצרים (tasks קלים)
- **עלות:** **חינם!**
- **Rate Limit:** 1,500 requests/day
- **מהירות:** מהיר מאוד

**קוד:** `backend/services/simpleAI.js` - `identifyProduct()`

---

#### **2. Anthropic Claude Sonnet 4** 🔵
- **שימוש:** בניית דוסייה עם Web Search
- **עלות:** בתשלום (~$0.03 per request)
- **Rate Limit:** 30,000 tokens/minute
- **תכונה מיוחדת:** Web Search Tool (חיפוש בזמן אמת!)

**קוד:** `backend/services/simpleAI.js` - `buildDossier()`

---

### **איך ה-Routing עובד:**

```javascript
// backend/services/simpleAI.js

// Task: identify_product → Gemini (חינם)
if (task === 'identify_product') {
  return await this.useGemini(task, data);
}

// Task: build_dossier → Claude (עם Web Search)
if (task === 'build_dossier' && this.claude) {
  return await this.useClaude(task, data);
}
```

**יתרונות:**
- ✅ חיסכון בעלויות (Gemini חינם ל-tasks קלים)
- ✅ איכות גבוהה (Claude ל-analysis מורכב)
- ✅ Web Search אמיתי (Claude)

---

### **Rate Limiting** ⏱️

**Claude:** 
- 2 שניות בין בקשות (כדי למנוע 429 errors)
- קוד: `backend/services/simpleAI.js` - `waitForClaude()`

**Gemini:**
- ללא rate limiting מיוחד (חינם, אבל יש limit יומי)

## 💾 Database Schema

### **טבלה: `products`**

```sql
CREATE TABLE products (
  id SERIAL PRIMARY KEY,
  name TEXT UNIQUE NOT NULL,
  category TEXT,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);
```

**תפקיד:** אחסון רשימת מוצרים שנותחו

---

### **טבלה: `dossiers`** (הטבלה המרכזית)

```sql
CREATE TABLE dossiers (
  id SERIAL PRIMARY KEY,
  product_id INTEGER UNIQUE REFERENCES products(id) ON DELETE CASCADE,
  
  -- Scores (0-100)
  overall_score INTEGER,
  quality_score INTEGER,
  value_score INTEGER,
  reliability_score INTEGER,
  
  -- Analysis (JSON)
  summary TEXT,
  pros JSONB,              -- ["pro 1", "pro 2", ...]
  cons JSONB,              -- ["con 1", "con 2", ...]
  common_failures JSONB,   -- ["issue 1", ...]
  best_for JSONB,          -- ["user type 1", ...]
  not_recommended_for JSONB, -- ["user type 1", ...]
  
  -- Metadata
  total_reviews INTEGER,
  confidence_score DOUBLE PRECISION,
  status TEXT,             -- 'ready', 'building', 'error'
  created_at TIMESTAMP DEFAULT NOW(),
  last_updated TIMESTAMP DEFAULT NOW()
);
```

**תפקיד:** אחסון הניתוח המלא של כל מוצר

---

### **טבלה: `reviews_summary`** (עתיד)

```sql
CREATE TABLE reviews_summary (
  id SERIAL PRIMARY KEY,
  product_id INTEGER UNIQUE REFERENCES products(id) ON DELETE CASCADE,
  sources_found INTEGER,
  overall_sentiment TEXT,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);
```

**תפקיד:** סיכום מקורות הביקורות (כרגע לא בשימוש פעיל)

## 🖥️ Frontend Flow

### **1. Search Page** (`/search`)

**קובץ:** `frontend/src/pages/SearchPagePremium.jsx`

**זרימה:**
```
1. משתמש מזין שם מוצר → query state
2. לחיצה על "Analyze" → handleSearch()
3. בדיקה: האם Backend ער? → wakeBackend()
4. GET /api/search?q=... → חיפוש בדאטאבייס
5. תוצאות מוצגות → results state
6. לחיצה על מוצר → navigate(/product/:id)
```

**תכונות מיוחדות:**
- **Wake Backend:** אם Render sleeping, מעיר אותו
- **Retry Logic:** אם נכשל, מנסה שוב אחרי 8 שניות
- **Deep Link Support:** `/search?q=iPhone 15` → auto-search

---

### **2. Dossier Page** (`/product/:id`)

**קובץ:** `frontend/src/pages/DossierPagePremium.jsx`

**זרימה:**
```
1. useEffect → useProductDossier(productId)
2. GET /api/products/:id → טעינת דוסייה
3. הצגת:
   • Scores (overall, quality, value, reliability)
   • THE VERDICT (BUY IF / DON'T BUY IF)
   • Pros & Cons
   • Data Sources
   • Summary
```

**קומפוננטה:** `frontend/src/hooks/useProductDossier.js`

---

### **3. Home Page** (`/`)

**קובץ:** `frontend/src/pages/Home.jsx`

**תכונות:**

### **4. API URL Detection** 🔍

הפרונטאנד מזהה אוטומטית את ה-API URL:

```javascript
const getAPIUrl = () => {
  // 1. Environment variable (עדיפות ראשונה)
  if (import.meta.env.VITE_API_URL) {
    return import.meta.env.VITE_API_URL;
  }
  
  // 2. Production (Vercel / clearpickai.com)
  const hostname = window.location.hostname;
  if (hostname === 'www.clearpickai.com' || 
      hostname === 'clearpickai.com' ||
```
