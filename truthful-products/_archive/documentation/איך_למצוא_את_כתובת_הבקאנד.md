# 🔍 איך למצוא את כתובת הבקאנד ב-Render

## שלב 1: היכנס ל-Render Dashboard
1. פתח: https://dashboard.render.com
2. התחבר לחשבון שלך

## שלב 2: מצא את השירות
1. לחץ על **"Web Services"** בתפריט השמאלי
2. חפש שירות בשם כמו:
   - `clearpick-backend`
   - `clearpick-ai`
   - `truthful-products-backend`
   - או כל שם אחר שהוגדר

## שלב 3: העתק את הכתובת
1. לחץ על השירות
2. בחלק העליון תראה כתובת כמו:
   ```
   https://clearpick-ai.onrender.com
   ```
   או
   ```
   https://clearpick-backend.onrender.com
   ```
   או שם אחר

## שלב 4: בדוק שהשרת עובד
פתח בדפדפן:
```
https://[הכתובת-שמצאת].onrender.com/api/health
```

אמור להופיע JSON עם סטטוס.

---

## ⚠️ אם אין שירות ב-Render

אם לא מוצא שירות, צריך ליצור אחד:

1. **לחץ "New +"** → **"Web Service"**
2. **חבר את ה-Repository** (GitHub/GitLab)
3. **הגדרות:**
   - Name: `clearpick-backend` (או שם אחר)
   - Build Command: `npm install`
   - Start Command: `npm start`
   - Environment: `Node`
4. **שמור**

הכתובת תוצג אחרי הפריסה הראשונה.
