## ⚡ Quick Start (2 דקות)

### הרץ את הפרויקט:

**Terminal 1 - Backend:**
```bash
cd truthful-products/backend
node server-unified.js
```
→ צפוי לראות: `🚀 Running on http://localhost:3000`

**Terminal 2 - Frontend:**

## 🎯 מה המערכת עושה?

**בקצרה:**
1. משתמש מחפש מוצר (למשל: "iPhone 15")
2. Backend אוסף ביקורות מ-**Reddit** (20-50 ביקורות)
3. AI מנתח ומסכם (**Gemini + Claude**)
4. המערכת מחזירה **דירוג + pros/cons + סיכום**
5. כל זה נשמר ב-**cache** (מהיר פי 100 בפעם הבאה!)

**זמן תגובה:**
- חיפוש ראשון: ~20 שניות
- חיפוש cached: ~50ms ⚡
