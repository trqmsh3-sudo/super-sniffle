## 🎯 סקירה כללית

**ClearPick.ai** היא פלטפורמת AI מתקדמת לאנליזה חכמה של מוצרים, המספקת המלצות רכישה מבוססות-מידע באמצעות שילוב של Gemini AI ו-Claude AI עם Smart Routing.

## 🧩 מבנה הפרויקט

### Backend Services (13 שירותים עיקריים)

#### **1. Smart AI Router** (`aiRouter.js`)
- **תפקיד:** מנתב בחכמה בין Gemini (חינם) ל-Claude (בתשלום)
- **חיסכון:** 70-80% מעלויות AI
- **לוגיקה:** משימות פשוטות → Gemini, מורכבות → Claude
- **שורות קוד:** ~595

#### **2. Dossier Builder** (`simpleDossierBuilder.js`)
- **תפקיד:** בונה דוסיאות מוצרים מקיפים
- **תהליך:** Reddit Scraping → Data Aggregation → AI Analysis → DB
- **פיצ'רים:** Smart Cache, Universal Images, Quality Monitor
- **שורות קוד:** ~441

#### **3. Headphones Wizard** (`headphonesWizard.js`)
- **תפקיד:** אשף המלצות לאוזניות
- **קריטריונים:** תקציב, שימוש, פיצ'רים, מותג
- **מקורות:** 4 subreddits (r/headphones, r/audiophile, וכו')
- **שורות קוד:** ~401

#### **4. Reddit Scraper** (`redditScraper.js`)
- **תפקיד:** גורד ביקורות אמיתיות מ-Reddit
- **API:** Reddit API integration
- **פילטרים:** Spam detection, relevance scoring

#### **5. Data Aggregator** (`dataAggregator.js`)
- **תפקיד:** מנתח sentiment, מזהה pros/cons
- **אלגוריתמים:** Pattern recognition, common issues detection
- **פלט:** Structured data for AI

#### **6. Google Shopping Service** (`googleShoppingService.js`)
- **תפקיד:** משווה מחירים מ-Google Shopping API
- **אינטגרציה:** Real-time price fetching

#### **7. Universal Image Service** (`universalImageService.js`)
- **תפקיד:** מביא תמונות מוצר מ-5 מקורות שונים
- **מקורות:** Google Images, Official Sites, Reddit, etc.
- **Fallback:** Multiple source redundancy

#### **8. Smart Cache** (`smartCache.js`)
- **תפקיד:** Redis caching עם TTL דינמי
- **לוגיקה:** TTL מבוסס confidence score
- **תכונות:** Build locks, stale detection

#### **9. Quality Monitor** (`qualityMonitor.js`)
- **תפקיד:** מוודא איכות דוסיאות
- **בדיקות:** Generic content detection, completeness checks

#### **10. Price Comparison** (`priceComparison.js`)
- **תפקיד:** משווה מחירים מאתרים ישראליים
- **אתרים:** Zap, KSP, Bug

#### **11. Product Image Service** (`productImageService.js`)
- **תפקיד:** Fallback service לתמונות
- **תכונות:** Image validation, size optimization

#### **12. Product Validator** (`productValidator.js`)
- **תפקיד:** וולידציה של נתוני מוצרים
- **בדיקות:** Schema validation, data integrity

#### **13. Simple AI** (`simpleAI.js`)
- **תפקיד:** Wrapper פשוט ל-AI services
- **מטרה:** Simplified interface for Gemini/Claude
