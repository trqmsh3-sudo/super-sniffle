## 🌐 **שלב 2: Deploy Frontend ל-Vercel**

### **למה Vercel?**
```
✓ Unlimited bandwidth! (לא כמו Netlify)
✓ Auto-deploy מ-GitHub
✓ Edge network (מהיר בכל העולם)
✓ Custom domains חינם
✓ HTTPS אוטומטי
✓ Preview deployments
```

### **הוראות:**

**2.1 - הכן את הקוד (2 דקות):**
```bash
cd frontend

# צור .env.production:
echo 'VITE_API_URL=https://clearpick-api.onrender.com/api' > .env.production

# בנה:
npm run build
# זה יצור: dist/
```

**2.2 - Deploy ל-Vercel:**

**דרך 1: CLI (מהיר!):**
```bash
# התקן Vercel CLI (פעם אחת):
npm install -g vercel

# Deploy:
vercel

# עקוב אחרי ההוראות:
? Set up and deploy? Yes
? Which scope? <your-account>
? Link to existing project? No
? Project name? clearpick
? Directory? ./
? Override settings? No

# Deploy production:
vercel --prod
```

**דרך 2: Dashboard (ידני):**
```
1. לך ל: https://vercel.com/dashboard
2. לחץ "Add New" → "Project"
3. Import Git Repository
4. בחר את ה-repo
5. Framework: Vite
6. Root Directory: frontend
7. Build Command: npm run build
8. Output Directory: dist
9. Environment Variables:
   VITE_API_URL=https://clearpick-api.onrender.com/api
10. לחץ "Deploy"
```

**תקבל URL:**
```
https://clearpick.vercel.app
```

## 🌍 **שלב 3: חבר את הדומיין שלך מ-Namecheap**

### **3.1 - ב-Vercel:**
```
1. Vercel Dashboard → Project "clearpick"
2. Settings → Domains
3. לחץ "Add"
4. הקלד את הדומיין שלך (למשל: clearpick.com)
5. לחץ "Add"

Vercel ייתן לך הוראות DNS:
→ Type: A Record
→ Name: @
→ Value: 76.76.21.21

→ Type: CNAME
→ Name: www
→ Value: cname.vercel-dns.com
```

### **3.2 - ב-Namecheap:**
```
1. לך ל: https://namecheap.com
2. Dashboard → Domain List
3. לחץ "Manage" ליד הדומיין שלך
4. Advanced DNS

5. הוסף/עדכן records:
   
   Type      Host    Value                   TTL
   ────────────────────────────────────────────────
   A Record  @       76.76.21.21            Automatic
   CNAME     www     cname.vercel-dns.com   Automatic

6. שמור
```

### **3.3 - המתן (5-30 דקות):**
```
DNS propagation לוקח זמן:
- 5 דקות (בד"כ)
- 30 דקות (מקסימום)
- 48 שעות (נדיר)

בדוק:
https://www.whatsmydns.net/#A/yourdomain.com
```

## 🔧 **אם משהו לא עובד:**

### **Backend לא עולה:**
```
1. בדוק logs ב-Render
2. ודא ש-Environment Variables נכונים
3. בדוק שPostgreSQL מחובר
```

### **Frontend לא מתחבר ל-Backend:**
```
1. בדוק VITE_API_URL ב-Vercel
2. ודא ש-CORS מוגדר נכון בBackend
3. בדוק Network tab בדפדפן
```

### **Domain לא עובד:**
```
1. בדוק DNS ב-Namecheap
2. המתן 30 דקות ל-propagation
3. בדוק ב-whatsmydns.net
```
