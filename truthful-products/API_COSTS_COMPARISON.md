## 🧮 **חישוב ROI:**

### **עם 1,500 משתמשים ליום:**

**הכנסות (משוערות):**
```
1,500 משתמשים × 10% CTR = 150 clicks
150 clicks × 5% conversion = 7.5 מכירות
7.5 מכירות × $20 עמלה = $150/יום
$150/יום × 30 ימים = $4,500/חודש
```

**עלויות:**
```
Gemini Free: $0/חודש

```

**רווח נקי:**
```
$4,500 - $0 = $4,500/חודש (Gemini)
$4,500 - $50 = $4,450/חודש (OpenAI)
```
