## 📝 Notes:

### API Keys Needed (All Free!):
1. **Unsplash:** https://unsplash.com/developers
   - 50,000 requests/month
   - Sign up → Create App → Copy Access Key

2. **Pexels:** https://www.pexels.com/api/
   - 200 requests/hour
   - Sign up → Generate API Key

3. **Bing Image Search:** https://azure.microsoft.com/en-us/services/cognitive-services/bing-image-search-api/
   - 1,000 transactions/month
   - Create Azure account → Create resource → Copy key

4. **Redis Cloud:** https://redis.com/try-free/
   - 30MB free tier
   - Create account → Create database → Copy connection URL
