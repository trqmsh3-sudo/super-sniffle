## 🚀 איך להריץ עכשיו?

### התקנה (פעם אחת):
```bash
# Backend
cd backend
npm install  # express-rate-limit כבר מותקן!

# Frontend
cd frontend
npm install  # הכל מותקן!
```

### הרצה:
```bash
# Terminal 1
cd backend
node server.js

# Terminal 2
cd frontend
npm run dev

# דפדפן
http://localhost:5173
```
